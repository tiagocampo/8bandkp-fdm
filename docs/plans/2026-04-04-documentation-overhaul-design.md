# Documentation Overhaul Design

## Goal

Replace stale documentation files (`modernization_plan.md`, `plan.md`, `checklist.md`,
outdated sub-READMEs) with a comprehensive lecture-style documentation suite.

Every equation maps to code. Every concept maps to a real plot generated by the code.
The docs read as a pedagogical progression from bulk k.p theory to quantum wires,
serving students learning the physics, users running the code, and developers extending it.

## Audience

1. **Students** learning k.p theory — full derivations, pedagogical explanations
2. **Users** who know semiconductor physics — what the code computes, how to configure it
3. **Developers** extending the code — where to hook in, module architecture

## Decisions

- **Format:** Pure Markdown with LaTeX math ($$...$$). No build tools (no FORD).
- **Figures:** PNG committed in `docs/figures/`, generated by a single Python script.
- **Examples:** Real runs with real configs and real output. No mock data.

## Chapter Content Outline

Each chapter follows this template:
1. **Theory** — derivation with LaTeX math
2. **In the code** — which module/subroutine implements it, key types
3. **Computed example** — input config, output data, plots
4. **Discussion** — physical interpretation, limitations, tips

### Ch 01 — Bulk Band Structure

- Bloch theorem, lattice periodicity
- k.p as perturbation expansion around Gamma point
- 8-band basis: Gamma_6 (CB s-like), Gamma_8 (HH/LH p-like), Gamma_7 (SO p-like)
- Mapping basis states 1-8 to code ordering (bands 1-4 valence, 5-6 SO, 7-8 CB)
- Kane matrix elements: P (conduction-valence coupling), Q/R/S/T (valence-valence)
- Explicit 8x8 Hamiltonian matrix
- Example: GaAs bulk, show 8-band E(k) along [100]
- Code: `ZB8bandBulk` in `hamiltonianConstructor.f90`, material params in `parameters.f90`

### Ch 02 — Quantum Well

- From bulk to confined: envelope function approximation
- Adding z-dependence, replacing kz with -i d/dz
- QW Hamiltonian as 8x8 block matrix, each block NxN (N=FDstep)
- Block structure: k.p terms (Q, R, S, T, P) + band offsets from `profile`
- Heterostructure: multiple materials, band alignment
- Type-I vs type-II alignment (InAs/GaSb example)
- Example: InAs/GaSb/AlSb type-II QW, show E(k_parallel) subbands
- Code: `confinementInitialization`, `constructHamiltonian` in `hamiltonianConstructor.f90`

### Ch 03 — Wavefunctions

- Eigenvectors of the 8N x 8N Hamiltonian
- Spatial decomposition: 8 components, each with N grid points
- Band character: |psi_b(z)|^2 and integrated parts
- Probability density per band vs total
- Heavy-hole vs light-hole character mixing at finite k
- Example: first few QW eigenstates, show wavefunction components
- Code: `writeEigenfunctions`, `writeEigenvalues` in `outputFunctions.f90`

### Ch 04 — Strain

- Lattice-mismatched epitaxy: a_substrate vs a_layer
- Biaxial strain tensor: eps_xx = eps_yy = (a_sub - a_layer)/a_layer
- Poisson response: eps_zz = -2 C12/C11 * eps_xx
- Bir-Pikus Hamiltonian: hydrostatic + shear contributions
- Strain-induced band edge shifts and HH/LH splitting
- 2D strain for wire geometry
- Example: strained heterostructure, show strain tensor and shifted bands
- Code: `strain_solver.f90`, deformation potentials in `parameters.f90`

### Ch 05 — g-Factor

- Zeeman Hamiltonian: magnetic field coupling to spin and orbital angular momentum
- Second-order Lowdin partitioning: project onto CB (or VB) subspace
- P and Q subspaces, effective 2x2 spin Hamiltonian
- g-tensor: g_eff = 2 + Delta_g (deviation from free electron)
- Inter-band and intra-band contributions to Delta_g
- Wire g-factor: k_z subband decomposition, all 3 spatial directions
- Example: QW CB g-factor, show g-components
- Code: `gfactor_functions.f90`, `compute_pele_2d`, `compute_gfactor_wire`

### Ch 06 — Optical Properties

- Inter-band optical transitions: momentum matrix elements <psi_i|p|psi_j>
- Oscillator strength: f = (2/m0) sum|p|^2 / Delta_E
- Polarization dependence: px, py, pz components
- Selection rules from 8-band basis (TE vs TM)
- Wire optical transitions: inter-subband CB-VB pairs
- Example: wire optical spectrum, polarization-resolved
- Code: `compute_optical_matrix_wire`, `compute_pele_2d` in `gfactor_functions.f90`

### Ch 07 — Self-Consistent Schrodinger-Poisson

- Coupling: charge from eigenstates -> Poisson -> potential -> Hamiltonian -> eigenstates
- Poisson equation: box-integration FD, variable dielectric
- Charge density: k_parallel sampling of occupied subbands (handles nonparabolicity)
- Fermi level: charge neutrality or fixed value
- Per-layer doping (ND, NA)
- Linear mixing warm-up -> DIIS/Pulay acceleration
- Boundary conditions: Dirichlet-Dirichlet, Dirichlet-Neumann
- Example: n-doped GaAs/AlAs QW, show band bending, charge, convergence
- Code: `poisson.f90`, `charge_density.f90`, `sc_loop.f90`

### Ch 08 — Quantum Wire

- 2D confinement in x-y plane, free propagation along z
- 2D FD grid: nx x ny points, Kronecker product structure
- Hamiltonian: 8 * Ngrid x 8 * Ngrid, sparse storage
- COO -> CSR conversion, PARDISO direct solver
- 2D Poisson solver for wire SC
- Wire g-factor and optical transitions (Ch 05/06 methods applied to wire)
- Example: GaAs rectangular wire, subbands, 2D density, g-factor, optical
- Code: `geometry.f90`, `sparse_matrices.f90`, wire path in `hamiltonianConstructor.f90`

### Ch 09 — Numerical Methods

- FD stencils: orders 2-10, coefficients, boundary handling
- Convergence: O(h^n) rate, practical tradeoffs
- Eigensolvers: dense zheevx (QW) vs sparse FEAST (wire)
- Sparse matrix formats: COO, CSR, memory layout
- OpenMP parallelization: k-point sweep, SpMV
- Example: convergence study with FD order, timing comparison
- Code: `finitedifferences.f90`, `eigensolver.f90`, `sparse_matrices.f90`, `linalg.f90`

### Ch 10 — Extending the Code

- Module dependency graph
- Data flow: input.cfg -> simulation_config -> Hamiltonian -> solver -> output
- Adding a new material: step-by-step in `parameters.f90`
- Adding new physics: where to hook into Hamiltonian construction
- Adding new output: the output pipeline
- Adding new tests: pFUnit and regression test templates
