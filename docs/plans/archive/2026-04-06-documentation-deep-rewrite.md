- **Chapter template:** Each chapter is a standalone markdown file with `# Chapter NN: Title` heading. Sections: `## 1. Theory`, `## 2. In the Code`, `## 3. Computed Example`, `## 4. Discussion` (or `## 4. Published Example`).
